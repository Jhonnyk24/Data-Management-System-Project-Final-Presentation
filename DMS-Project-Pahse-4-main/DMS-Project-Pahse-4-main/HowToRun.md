# Movie Database System

## How to Run

1. Install **Java 17** or higher.
2. Install **MySQL 8.x** and create a database named `moviesdb`.
3. Import the provided database script (see below).
4. Open the project in IntelliJ IDEA.
5. Add **MySQL Connector/J JAR** to the project libraries:
   - File → Project Structure → Libraries → + → select `mysql-connector-j-9.5.0.jar`
6. Run `Main.java` to start the program.
7. Connect to the database using:
   - Host: `localhost`
   - Database: `moviesdb`
   - User: `root`
   - Password: `your_password`
